/*******************************************************************/
ダッシュボードプラグイン for question2answer
/*******************************************************************/

/*-----------------------------------------------------------------*/
1. 概要
/*-----------------------------------------------------------------*/
このパッケージはquestion2answer用のプラグインです。

question2answer: http://www.question2answer.org/

/*-----------------------------------------------------------------*/
2. このプラグインの機能
/*-----------------------------------------------------------------*/
1.このプラグインはQ2Aにダッシュボードページを追加できます。
2.ダッシュボードに表示するコンテンツ（３ブロック）をオプションで変更できます。
3.ダッシュボードのデザインをjQuery UIまたはカスタムCSSで変更できます。
4.ユーザーがログイン後にダッシュボードページにリダイレクトできます。

/*-----------------------------------------------------------------*/
3. 対応バージョン
/*-----------------------------------------------------------------*/
question2answer V1.5以上

/*-----------------------------------------------------------------*/
4. インストール／設定
/*-----------------------------------------------------------------*/
1.アーカイブファイルを解凍します。
2.qa-pluginフォルダの下にdashboardフォルダをアップします。
3.管理者権限でログインします。
4.「管理」->「プラグイン」メニューを選択します。
5.オプションを設定してから保存します。
6.「管理」->「ページ」メニューを選択します。
7.「リンクを追加」でダッシュボードページへのリンクを追加します。

/*-----------------------------------------------------------------*/
5. アンインストール
/*-----------------------------------------------------------------*/
1.管理者権限でログインします。
2.「管理」->「プラグイン」メニューを選択します。
3."dashboard" のオプションをクリックします。
4."dashboard" の"初期値に戻す"をクリックします。
5.qa-pluginフォルダの下のdashboard フォルダを削除します。

/*-----------------------------------------------------------------*/
6. オプション説明
/*-----------------------------------------------------------------*/
[ブロックXを有効にする]
ダッシュボードのブロックXが表示されます。

[ブロックXの名前]
ダッシュボードのブロックXへのページ内リンクの名前を指定します。

[ブロックXのコンテンツ - HTML可]
ダッシュボードのブロックXのコンテンツをHTML形式で入力します。

[jQuery UIを有効にする]
ダッシュボードがjQuery UIのタブ形式で表示されます。

[jQuery UIのスタイルシートのパス]
jQuery UIのスタイルシートへのパスをQ2Aルートからの相対パスで指定します。

[カスタムスタイルシートのパス]
カスタムスタイルシートへのパスをQ2Aルートからの相対パスで指定します。

[リダイレクトを有効にする]
ダッシュボードプラグインによるリダイレクト機能が有効になります。

[リダイレクトアドレス（Q2Aルート相対）]
ユーザーがログインした後にリダイレクトするダッシュボードページのパスを指定します。

/*-----------------------------------------------------------------*/
7. 追加情報
/*-----------------------------------------------------------------*/

[コンテンツの編集]
WYSIWYG Editor for adminプラグインをインストールすればダッシュボードのコンテンツを
WYSIWYGエディタで編集できます。
http://www.question2answer.org/qa/21275/new-plugin-wysiwyg-editor-for-admin-beta
追加するID）
,dashboard_block1_content_field,dashboard_block2_content_field,dashboard_block3_content_field

[jQuery-UIのデザイン]
テーマローラーを使ってダッシュボードのデザインを簡単に変更できます。
下のサイトで作成したテーマファイルをサーバーにアップロードします。
”jQuery UIのスタイルシートのパス”にアップロードしたファイルを指定します。
http://jqueryui.com/themeroller/
